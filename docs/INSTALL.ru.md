# Установка Gaiare Studio

Инструкция для дизайнеров.

> ⚠️ **Приложение пока без цифровой подписи.** Поэтому при первом запуске
> Windows и macOS показывают предупреждение системы безопасности — это **не
> вирус**, так система реагирует на любые программы без платного сертификата
> разработчика. Нужно один раз пройти шаги ниже.
>
> ✅ **Ручную установку нужно сделать только один раз.** Начиная с этой версии
> приложение обновляется само — следующие версии установятся автоматически
> прямо в окне программы (см. раздел [Обновления](#обновления)).

---

## Windows

1. **Скачайте установщик.** На странице релиза (репозиторий → **Releases**) в
   разделе **Assets** скачайте файл вида:
   - `Gaiare Studio_X.Y.Z_x64-setup.exe` (рекомендуется), либо
   - `Gaiare Studio_X.Y.Z_x64_en-US.msi`

2. **Запустите** скачанный файл двойным кликом.

3. **Пройдите предупреждение SmartScreen.** Появится синее окно с текстом
   *«Система Windows защитила ваш компьютер»*:
   - нажмите **«Подробнее»** (More info);
   - затем нажмите кнопку **«Выполнить в любом случае»** (Run anyway).

4. **Завершите установку** — следуйте шагам мастера (кнопки «Далее» / Install).

5. **Запустите приложение.** После установки оно открывается обычным двойным
   кликом, предупреждение больше не появляется.

---

## macOS

> ⚠️ Сборка для macOS работает только на компьютерах с процессором **Apple
> Silicon (M1/M2/M3 и новее)**. На старых Mac с процессором Intel она не
> запустится.

1. **Скачайте образ.** На странице релиза в разделе **Assets** скачайте файл
   вида `Gaiare Studio_X.Y.Z_aarch64.dmg`.

2. **Откройте `.dmg`** и перетащите иконку **Gaiare Studio** в папку
   **Программы** (Applications).

3. **Первый запуск.** При двойном клике появится сообщение, что приложение
   *«не удаётся открыть, так как Apple не может проверить его на наличие
   вредоносного ПО»*. Это ожидаемо — закройте окно.

4. **Разрешите запуск:**
   - откройте **Системные настройки → Конфиденциальность и безопасность**;
   - пролистайте вниз до строки *«Программа "Gaiare Studio" заблокирована…»*;
   - нажмите **«Всё равно открыть»** (Open Anyway), затем **«Открыть»** (Open).
   - На старых версиях macOS можно вместо этого: правый клик (или Ctrl+клик) по
     приложению → **«Открыть»** → **«Открыть»**.

5. **Готово.** Дальше приложение запускается обычным способом без предупреждений.

## Обновления

Начиная с этой версии приложение **обновляется само**. Когда выходит новая
версия, при запуске (или через **Настройки → About → Check for updates**)
появляется окно с кнопкой **«Download & Restart»**. Нажмите её — приложение само
скачает обновление и перезапустится в новой версии.

Скачивать `.dmg`/`.exe` и переустанавливать вручную больше не нужно — это
требуется только для самой первой установки.
